# [🍇 포동포동 : 포도나무 감상 걸렸네](podong-podong.vercel.app)
> 포동포동은 책을 읽고 감상평을 작성해 포도알을 채울 수 있는 독서 장려 애플리케이션입니다.
> 이 프로젝트는 '테오의 스프린트'를 참여, 구글 스프린트 진행 방식을 직접 배워보며 만든 사이드 프로젝트입니다.

<a href="podong-podong.vercel.app">
</a>

[테오의 스프린트 17기 8조: 포동포동]
- 🍇 포도 등록: 책을 검색해 등록하면 포도 하나가 농장에 등록됩니다. 하나씩 모아가며 포도농장을 키워보세요! 
- 📌 메인 포도 등록: 현재 읽고 있는 책을 포도농장의 메인 포도로 등록하세요. 
- ✨ 포도알 관리: 감상평을 등록하거나 농도를 지정하면 포도알이 채워집니다. 비어있는 포도알을 포동포동하게 만들어보세요.
- ✏️ 포도알 감상평 확인: 등록된 포도(책)의 감상평은 메인 포도의 포도알을 클릭해 확인 가능합니다. 
- 🌳 포도농장(서재) 방문: 포도농장을 방문하면 당신이 키운 포도들을 한 눈에 확인 할 수 있습니다. 어떤 포도들이 당신의 독서 농장을 장식하고 있나요?

포도농장과 함께라면 독서가 더욱 즐거워집니다! 함께 독서의 즐거움을 나눠보세요.
